# Site for hangjs 2014

## Pre-requisites

    nodejs 0.10.x

    ruby compass gem

## How to

### Install nodejs modules
```bash
make i
```

### Install bower packages
```bash
make boweri
```

### Watch files changes and start development server
```bash
make
```

### Watch files changes and start development server and open page in default browser
```bash
make browser
```
